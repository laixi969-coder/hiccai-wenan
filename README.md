# hiccai-wenan

蔡蔡的商业文案专家 Skill（Claude Code）。把产品翻译成生活，把卖点翻译成购买理由，把洞察压缩成金句，把高端品牌写出分寸与秩序。

## 能力范围

- 品牌定位 / Slogan / 品牌金句
- 销售转化文案 / 社交媒体海报 / 促销节点
- 小红书标题、封面标题、正文
- 朋友圈 / 短视频脚本 / 品牌片旁白
- 电商详情页 / 招商话术
- 地产 / 豪宅 / 高端品牌文案
- 奢侈品 / 极致高端（珠宝、腕表、高定、游艇、私人会所、高端酒店、艺术收藏）
- 洞察转创意 / 创意概念 / 海报画面 / 15 秒短视频脚本
- 整合传播创意 / AI Agent 角色提示词

## 核心特性

- **四层级文案模型**：属性 → 功能 → 心理 → 价值
- **商业转化链**：处境 → 痛点 → 价值 → 判断 → 身份 → 购买理由 → 行动
- **洞察转创意系统**：现象洞察 → 事故感 → 表达变形 → 品牌转折 → 创意输出
- **最终输出洞察门槛**：所有海报、标题、脚本、Slogan、金句输出前，必须先通过"对啊，就是这样"的洞察标准
- **奢侈品独立表达模型**：材质、工艺、时间、留白、距离感，不写"贵"写时间成本
- **平台适配**：小红书 / 朋友圈 / 海报 / 详情页 / 短视频，各有语言系统
- **价值观边界**：小红书禁低俗、禁焦虑制造、禁虚假承诺
- **通用 + 奢侈品双套禁用词表**：自动替换"高级感拉满 / 尊享 / 沉浸式"等空话
